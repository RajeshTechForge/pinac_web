## ðŸš€ Getting Started

1. Clone the Repository

   ```bash
   git clone https://github.com/pinacai/pinac_web.git && cd pinac_web
   ```

2. Install Node dependencies

   ```bash
   npm install
   ```

3. Start the App
   ```bash
   npm run dev
   ```
